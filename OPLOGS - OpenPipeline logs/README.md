# OPLOGS - OpenPipeline logs

End-to-end guidance for managing logs with Dynatrace OpenPipeline.

## Structure
- NOTEBOOKS/ — Dynatrace notebook JSON files for each topic
- PDFs/ — Printable versions of the notebooks
- markdown/ — Markdown exports of the notebooks

## Notebook Lineup
1. [Fundamentals](markdown/-OPLOGS-01-fundamentals.md) — Core OpenPipeline concepts
2. [Migration](markdown/-OPLOGS-02-migration.md) — Moving from other logging platforms
3. [Pipeline Processing](markdown/-OPLOGS-03-pipeline-processing.md) — Transformations and processing stages
4. [Buckets & Governance](markdown/-OPLOGS-04-buckets-governance.md) — Storage management and governance
5. [Querying & Parsing](markdown/-OPLOGS-05-querying-parsing.md) — Query techniques and parsing rules
6. [Topology](markdown/-OPLOGS-06-topology.md) — Log topology and relationships
7. [Analytics](markdown/-OPLOGS-07-analytics.md) — Advanced log analytics
8. [Security](markdown/-OPLOGS-08-security.md) — Security, masking, and compliance

## Usage
1. Choose a format: import JSON from NOTEBOOKS/, read PDFs/ for print, or view markdown/ for lightweight browsing.
2. Start with Fundamentals before progressing to advanced topics.
