# OTEL - OpenTelemetry

Integrating OpenTelemetry with Dynatrace for observability.

## Structure
- NOTEBOOKS/ — Dynatrace notebook JSON files
- PDFs/ — Printable versions of each notebook
- markdown/ — Markdown exports of the notebooks

## Notebook Lineup
1. Fundamentals — Core OpenTelemetry concepts
2. Collector Architecture — Understanding the OTel Collector
3. Collector Deployment — Deploying and configuring collectors
4. Trace Instrumentation — Instrumenting applications for traces
5. Metrics Instrumentation — Instrumenting applications for metrics
6. Logs Instrumentation — Instrumenting applications for logs
7. Dynatrace Integration — Sending OTel data to Dynatrace
8. Troubleshooting — Diagnosing OpenTelemetry issues

## Usage
1. Choose a format: import JSON from NOTEBOOKS/, read PDFs/ for print, or view markdown/ for lightweight browsing.
2. Start with Fundamentals, then follow the sequence based on your instrumentation needs.
