# AUTOM - Dynatrace Automation

Comprehensive guide to automating Dynatrace configuration and operations.

## Structure
- NOTEBOOKS/ — Dynatrace notebook JSON files
- PDFs/ — Printable versions of each notebook
- markdown/ — Markdown exports of the notebooks

## Notebook Lineup
1. Automation Landscape — Overview of Dynatrace automation options
2. Settings API — Working with the Settings 2.0 API
3. Monaco — Configuration as code with Monaco
4. Terraform — Infrastructure as code with Terraform provider
5. Workflows — Dynatrace Workflows automation
6. SDKs — Using Dynatrace SDKs for automation
7. CI/CD Integration — Integrating Dynatrace into CI/CD pipelines
8. Migration Automation — Automating configuration migrations

## Usage
1. Choose a format: import JSON from NOTEBOOKS/, read PDFs/ for print, or view markdown/ for lightweight browsing.
2. Start with Automation Landscape for an overview, then explore specific tools as needed.
