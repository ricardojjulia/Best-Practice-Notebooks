# IAMADM - IAM Administration

Enterprise identity and access management administration for Dynatrace.

## Structure
- NOTEBOOKS/ — Dynatrace notebook JSON files
- PDFs/ — Printable versions of each notebook
- markdown/ — Markdown exports of the notebooks

## Notebook Lineup
1. Governance Foundations — IAM governance principles and strategy
2. SSO & Authentication — Single sign-on and authentication setup
3. Group Architecture — Designing group structures
4. Policy Authoring — Writing and managing policies
5. Boundary Design — Defining access boundaries
6. User Lifecycle — Managing user provisioning and deprovisioning
7. Audit & Compliance — Auditing access and ensuring compliance
8. Multi-Environment — IAM across multiple environments
9. Troubleshooting — Diagnosing and resolving IAM issues

## Usage
1. Choose a format: import JSON from NOTEBOOKS/, read PDFs/ for print, or view markdown/ for lightweight browsing.
2. Start with Governance Foundations, then progress through the series based on your needs.
